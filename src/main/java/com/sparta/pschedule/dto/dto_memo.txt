# DTO
1. 기존클래스 + Lombok vs record
java 14미만: 기존클래스 + Lombok
java 14이상: record (거의 코틀린 'data class' 와 유사해보임)

2. Validation
왠만하면 request(클라이언트로부터 들어온 데이터)만 검증을 하는것이 좋다.

3. 기본생성자
클라이언트가 보내는 JSON 데이터는
스프링 내부에서 Jackson이라는 라이브러리가 자바 객체로 변환해 줍니다(역직렬화)
- Jackson 내부구조
기본적으로 기본 생성자를 사용해서 객체를 먼저 만든 다음,
Getter나 Setter 또는 리플렉션을 통해 필드 값을 채워 넣습니다.
    => 반드시 @NoArgsConstructor 필수!!
    => Response는? 직렬화에는 기본생성자를 사용하지않아 필수는 아니다!!
